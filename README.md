# 🌐 Cloud Integrated Hybrid Network Project (Azure)

## 📌 Overview

This project demonstrates a hybrid network connecting an on-premises infrastructure with Microsoft Azure using secure communication.

## 🎯 Objectives

* Design a hybrid network architecture
* Connect on-prem network to Azure cloud
* Implement secure communication using VPN
